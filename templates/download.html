KidsCode Link

A small helper app that connects your KidsCode lessons to your Arduino board. Install it once — it runs quietly in the background whenever you need it.

⬇ Download KidsCode Link

Free · Windows 10 / 11 · No account required

What does KidsCode Link do?

  • 🌐 Bridges your browser and your Arduino — so when you click Upload in your lesson, the code goes straight to your board.
  • 🔒 Only talks to KidsCode. It doesn't send data anywhere else and doesn't run unless KidsCode needs it.
  • 💤 Stays in the system tray and wakes up automatically when you open a lesson with an Arduino step.

System requirements

Windows Windows 10 or Windows 11 (64-bit)
Browser Chrome or Edge (Chromium-based) — Firefox and Safari are not supported for uploads
Hardware Arduino UNO (or compatible board)
Cable USB-A to USB-B cable (the square plug)

⚠ Mac is not supported yet. Mac support is coming — check back!

How to install

  1. Click the Download button above and run the installer.
  2. Windows may show a security warning — click More info then Run anyway. This is normal for new apps.
  3. Follow the installer steps (just click Next → Install → Finish).
  4. KidsCode Link will appear in your system tray (bottom-right corner of your screen).
  5. Come back to your lesson and try uploading — it should connect automatically.
To uninstall: Open Windows Settings → Apps → search for "KidsCode Link" → Uninstall. Nothing else is left behind.
