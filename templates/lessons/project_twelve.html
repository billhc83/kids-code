🚓 Welcome to Night Patrol Academy

The sun has gone down. The streets are darker now. 🌃

At night, emergency vehicles must be easy to see. That is why they use light bars.

Tonight, you are training in the Emergency Systems Lab. Before a patrol vehicle goes on duty, the light bar must pass inspection. That is your job today.

🎯 Your Mission

This patrol vehicle has four parts:

  • 🔘 A Master Power Button
  • 🔴 A Red Light
  • 🔵 A Blue Light
  • ⚪ A Clear Strobe Light

The system follows simple rules:

  • If the Master Button is not pressed → all lights stay OFF.
  • If the Master Button is pressed → the light pattern begins.
  • The lights flash one at a time using 150ms delays.

Your job is to build the system and program the pattern.

🔎 How to Build the Logic

1️⃣ Create Your Light Variables

Before the lights can flash, the Arduino needs to know which pins control each light. Create variables like buttonPin, redLED, blueLED, and clearLED.

2️⃣ Set the PinModes

In the Setup section, tell the Arduino if each pin is an INPUT_PULLUP (for the button) or an OUTPUT (for the lights).

3️⃣ Follow the Pattern

The Academy uses a timing rule: Each flash and each pause must last 150 milliseconds. Use delay(150) to make the pattern clear!
