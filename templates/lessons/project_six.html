📜 The Navigation Script

void setup() {
  Serial.begin(9600);
  Serial.println("--- SUBMARINE POWER ON ---");
}

void loop() {
  int oceanLight = analogRead(A0);

  Serial.print("Light Level: ");
  Serial.println(oceanLight);

  if (oceanLight > 400) {
    Serial.println("☀️ SURFACE: I see dolphins! 🐬");
  } else {
    Serial.println("🐙 MIDNIGHT ZONE: Watch for tentacles! ✨");
  }

  delay(600);
}

🧬 How the Sonar Works

analogRead(A0)
Reads the Ocean Light — a number between 0 (Pitch Black) and 1023 (Super Bright). 🔢🌊

The "if" Dive-Line
If the light drops below 400, the submarine knows it is in deep dark water! 🔦

Serial.println()
Sends messages from the bottom of the ocean back to your computer screen! 🖥️📡
