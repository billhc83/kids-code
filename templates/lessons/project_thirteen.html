🔎 How Our Timer Works

In this project, we are learning how a program can measure time using a special function: millis().

⏱️ The Logic

  • First Press: The Arduino remembers the start time: startTime = millis().
  • Second Press: It checks the time again and subtracts the start: time = millis() - startTime.

✅ Remember

1000 milliseconds = 1 second. Try to stop your timer at exactly 5000 milliseconds! ⏱️
