💻 The Smart Logic

int lightSensor = A0;
int nightLight = 13;

void setup() {
  pinMode(nightLight, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int brightness = analogRead(lightSensor);
  Serial.println(brightness);

  if (brightness < 300) {
    digitalWrite(nightLight, HIGH);
    Serial.println("It's dark! Light ON 🌙");
  } else {
    digitalWrite(nightLight, LOW);
    Serial.println("Sun is up! Light OFF ☀️");
  }

  delay(100);
}

🧬 How it Works

Input vs Output
The sensor reads light (input), the LED reacts (output).

If / Else Logic
If it's dark → light turns ON.
Otherwise → light stays OFF.

Threshold
The number 300 decides when it is "dark".
