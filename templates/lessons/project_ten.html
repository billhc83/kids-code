🧠 Spy Logic Secret: We use Variables to store a "Status." boltA = 0 means stuck; boltA = 1 means open! 🔓

📜 The Security Script

// 🔐 Our Security Lock Variables
int LockA = 0; 
int LockB = 0;

void setup() {
  pinMode(2, INPUT_PULLUP);
  pinMode(3, INPUT_PULLUP);
  pinMode(8, OUTPUT);
  Serial.begin(9600);
  Serial.println("--- VAULT SYSTEM ARMED ---");
}

void loop() {
  LockA = digitalRead(2);
  LockB = digitalRead(3);

  if (LockA == 1 && LockB == 1) {
    digitalWrite(8, HIGH);
    Serial.println("🔓 ACCESS GRANTED: Vault Open!");
  } 
  else {
    digitalWrite(8, LOW);
    Serial.println("🔒 ACCESS DENIED: Locks Engaged.");
  }
  delay(500); 
}

🧬 How the Spy Logic Works

The "&&" Code
This means "AND." The computer asks: "Is Lock A open AND is Lock B open?" 🤝

Variables as Memory
Variables "remember" the switch state while the computer checks other things. 🧩

Decision Making
Engineers use variables to help computers make complex choices! 🧠
