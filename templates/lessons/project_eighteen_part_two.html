Build Your Speed Trap 🔧

Time to set up your speed trap! 🏎️ You will place TWO ultrasonic sensors a known distance apart. When something passes Sensor 1, the timer starts. When it reaches Sensor 2, the timer stops — and your Arduino does the math!

🧠 How the System Works

  • 📡 DETECT → Sensor 1 sees the object zoom past
  • ⏱️ START → The timer kicks off instantly
  • 📡 DETECT → Sensor 2 catches the object at the far end
  • ⏹️ STOP → The timer freezes
  • ➗ CALCULATE → Distance ÷ Time = Speed!
  • 📺 DISPLAY → Your speed appears on the Serial Monitor

🔁 The Flow of Your Program

Sensor 1 Detects 📡 → Timer Starts ⏱️ → Sensor 2 Detects 📡 → Timer Stops ⏹️ → Speed Calculated ➗ → Speed Displayed! 🏁
