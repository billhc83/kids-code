🚗 Engineer: Smart Backup System

Welcome, Engineer. You are building a Smart Backup Alarm for vehicle safety.

Your system will use a sensor to "see" objects and warn the driver before they crash. Every step you complete adds a new layer of safety.

🧠 How the System Works

Every Backup Alarm follows the same process:

  • 📡 MEASURE → The sensor sends a sound wave and waits for the echo
  • 📏 CALCULATE → The program converts time into distance
  • 🧠 DECIDE → The system checks if the object is Safe, Warning, or Danger
  • 🚨 ALERT → LEDs and Buzzers warn the driver

🔁 The Flow of Your Program

Your program will always follow this loop:

SEND SIGNAL → LISTEN FOR ECHO → CALCULATE DISTANCE → CHECK ZONES → ALERT DRIVER
