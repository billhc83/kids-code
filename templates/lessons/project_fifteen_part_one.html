🚗 Project: Smart Backup Alarm

Welcome to the Junior Engineer Training Program!

One day, your parent accidentally backs into your shiny new bike… or maybe even the neighbor’s car 😱. You realize something must be done.

As the innovative engineer you are, you decide to design a **smart backup alarm system** that warns when a car is too close to obstacles.

Your system will use a distance sensor to detect objects behind the car and alert the driver using LEDs and a buzzer.

The closer the object, the louder or faster the alert — just like a real car backup system!

🛠 What You Will Learn

  • How to read sensor data using an HC-SR04 ultrasonic sensor
  • How to control LEDs and a buzzer based on input
  • How to structure your code using conditional logic
  • How to work with loops to continuously check sensor input
