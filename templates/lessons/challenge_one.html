Calling all Junior Detectives! 🕵️‍♀️🔍

Someone has built a circuit, but the code has gone missing! Your job is to write a script that wakes up the system.

The Mission:

  1. Look at the Circuit Blueprint below and build the circuit on your breadboard.
  2. Use the Block Builder (the blue button in the bottom-right corner) to write your code.
  3. Make the Red Warning Light (LED) blink on and off like a heartbeat! ❤️⚡
  4. When you're done, close the Block Builder — your code will appear in the submission box automatically.
  5. Hit Submit to send it to the Head Detective for review! 🚀

🧱 Mystery Blueprint

Follow these coordinates to build the crime scene!

💡 Warning Light (LED)

  • Long leg (+): row 12, column E
  • Short leg (–): row 11, column E

⚡ The Helper (220 Ohm Resistor)

  • Leg 1: row 11, column D
  • Leg 2: row 7, column D

🧶 The Wires

  1. Pin GND → row 7, column E
  2. Pin 8 → row 12, column A

📝 The Goal

Make the LED turn ON for 1 second, then OFF for 1 second.

It should keep doing this forever so everyone knows the castle is safe! 🏰✨

Hint: The LED is on Pin 8.

🔵 Using the Block Builder

Click the blue button in the bottom-right corner to open the Block Builder. Use it to write your Arduino sketch — drag and drop blocks to build your code! When you're happy with your sketch, click the blue button again to close and your code will automatically appear in the submission box below.

💡 You can also click "📋 Grab from Builder" at any time to pull your latest code.

🔍 Need a Detective's Hint? (Click to open)

💡 Hint 1: Remember to use pinMode and OUTPUT in your setup so the light knows it needs to glow!

💡 Hint 2: Use digitalWrite and HIGH to turn the light on and LOW to turn it off.

💡 Hint 3: Don't forget a delay(1000); between your commands so the light has time to wake up and sleep! 😴

💡 Hint 4: The LED is connected to Pin 8.
