How to Move Your Code 🔌

Copy it: 📋
Find the code box and click the Copy button.

Open Arduino: ♾️
Open the Arduino app. It is the blue circle icon.

Clean the page: 🧹
Hold Ctrl and tap A, then hit Backspace.

Paste it: 📥
Hold Ctrl and tap V. Your code will pop up!

Send it: 🚀
Plug your board in. Click the Arrow button (➔). When it says "Done" your code is ready!
